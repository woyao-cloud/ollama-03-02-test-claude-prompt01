### 关键注意事项
1. **安全第一**: 所有用户输入必须验证，密码必须加密存储
2. **类型安全**: 使用Java Records定义DTO，确保类型安全
3. **测试驱动**: 先写测试，再实现功能
4. **分层架构**: 严格遵循Controller → Service → Repository → Entity分层
5. **审计追踪**: 所有敏感操作必须记录审计日志
6. **性能考虑**: 支持10,000 TPS，API响应时间<200ms
